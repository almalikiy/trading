# Android Trade Alert App

This folder will contain the source code for an Android application that provides sound alerts/notifications when a trade needs to be opened or closed. The app will support an always-on-top floating window with a small size for quick access.

## Planned Features
- Sound alert/notification for open/close trade signals
- Floating window (always on top, resizable)
- Minimal UI for quick action
- Integration with backend (REST/WebSocket)

## Tech Stack
- Android (Kotlin, Jetpack Compose or XML UI)
- Foreground service for always-on-top window
- Notification and sound management

## Getting Started
- Place all Android app source code in this folder.
- See README for build and usage instructions.
